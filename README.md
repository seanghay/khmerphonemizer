## Khmer Phonemizer

A Free, Standalone and Open-Source Khmer Grapheme-to-Phonemes.

### Installation

```shell
pip install khmerphonemizer
```

### Usage

```python
from khmerphonemizer import phonemize

text = "សួស្ដីមនុស្សខ្មែរ"
result = phonemize(text)
# =>
```
